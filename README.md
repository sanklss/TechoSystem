TechnoSystem - Управление IT-тарифами и заявками
WPF-приложение для IT-компании "ТехноСистемы"

Функциональность
- Авторизация пользователей (администратор, менеджер, клиент, гость)
- Просмотр и фильтрация тарифов на IT-услуги
- Управление заявками на подключение тарифов
- Учет свободных лицензий
- Проверка доступности лицензий при оформлении заявок

Технологии
- .NET 8.0 WPF
- Entity Framework Core
- MSSQL LocalDB
- xUnit для тестирования

Структура базы данных
User (пользователи)

Role (роли)

Tarif (тарифы)

Service (услуги)

Request (заявки)

RequestStatuse (статусы заявок)

Payment (платежи)

PaymentStatuse (статусы платежей)

PaymentWays (способы оплаты)
Запуск проекта

Предварительные требования
- Visual Studio 2022
- .NET 8.0 SDK
- MSSQL LocalDB

Шаги для запуска
1. Клонировать репозиторий
2. Открыть решение `GlobusApp.sln`
3. Восстановить пакеты NuGet
4. Запустить приложение
